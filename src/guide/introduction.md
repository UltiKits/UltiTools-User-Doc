# 简介

UltiTools 是一款面向 Minecraft Paper 服务器的模块化基础插件框架。

## 什么是 UltiTools？

UltiTools 最初发布于 2020 年，经过多次重构，现已发展为 v6.2.x 版本。它采用 **核心 + 模块** 的架构：核心框架负责模块加载、数据存储和配置管理，所有功能（基础指令、聊天、菜单、经济等）以独立模块的形式按需安装。

## 特点

- **模块化**：16+ 官方模块，按需安装，不臃肿不冗余
- **UPM 包管理器**：通过 `/upm install` 一键安装和更新模块
- **多存储后端**：支持 JSON、SQLite、MySQL 三种数据存储方式
- **云端面板**：通过 [panel.ultikits.com](https://panel.ultikits.com) 远程管理服务器
- **多语言**：内置中英文支持，可自定义语言文件
- **完全免费**：所有功能均为免费开源

## 系统要求

| 项目 | 要求 |
|------|------|
| Minecraft 版本 | 1.21 及以上 |
| 服务端 | Paper（推荐）或 Paper 分支 |
| Java 版本 | Java 21 |
| 可选前置 | Vault、PlaceholderAPI、LuckPerms |
